## ✨ About the Project

**Digital Tools** is a fully responsive single-page e-commerce website designed for selling premium digital productivity tools. It features a clean, modern UI with smooth interactions, a fully functional shopping cart, and an intuitive user experience that matches high-end SaaS platforms.

The design is fully inspired by professional Figma mockups and implemented with pixel-perfect accuracy.

---

## 🛠️ Technologies Used

- **React.js** (with Hooks & Functional Components)
- **Tailwind CSS** + **DaisyUI** (for beautiful components)
- **React Toastify** (for elegant notifications)
- **Lucide React** (for modern icons)
- **JavaScript (ES6+)**
- **JSON** (for product data management)

---

## 🔥 Key Features

### 1. Smart Shopping Cart System
- Add products with beautiful toast notifications
- Real-time cart count in navbar
- Remove items & total calculation
- "Proceed to Checkout" functionality

### 2. Beautiful Product Showcase
- 9 premium product cards in responsive 3-column grid
- Dynamic tags (`New`, `Popular`, `Best Seller`)
- Multiple pricing options (Free, Monthly, Yearly, One-time)

### 3. Smooth User Experience
- Toggle between Products & Cart section
- Fully responsive design (Mobile + Desktop)
- Professional 3-step getting started guide
- Clean and transparent pricing section
